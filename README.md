# Satellite Sim Interface

En React-applikasjon bygget med Next.js-rammeverket.

## Prosjektstruktur

```
satellite-sim-interface/
├── app/              # Next.js App Router
│   ├── layout.tsx    # Root layout
│   ├── page.tsx      # Hjemmeside
│   └── globals.css   # Global styling
├── package.json
├── tsconfig.json
└── next.config.js
```

## Installasjon

```bash
npm install
```

## Kjøring

### Utviklingsmodus

```bash
npm run dev
```

Applikasjonen kjører på http://localhost:3000

### Produksjonsbygge

```bash
npm run build
npm start
```

## Teknologier

- **Framework**: Next.js 14
- **UI**: React 18
- **Language**: TypeScript
